This project is a simple Hello World program created as part of my personal software portfolio for CSE 310. The purpose of this project is to demonstrate that my development environment is set up correctly and that I understand the basic workflow of creating, documenting, and publishing software using GitHub.

## Instructions for Build and Use

[Software Demo] https://youtu.be/Cz3Z9gf7Q5w

Steps to build and/or run the software:

1. Open a terminal in the project folder.
2. Run the program using the command `python3 hello_world.py`.

Instructions for using the software:

1. Run the program.
2. View the output printed to the terminal.
